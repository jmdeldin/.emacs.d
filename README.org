* Installation

You can clone directly as =$HOME/.emacs.d=:

#+BEGIN_SRC sh
$ git clone git://github.com/jmdeldin/.emacs.d.git ~/.emacs.d/
#+END_SRC

* Uninstallation

#+BEGIN_SRC sh
$ rm ~/.emacs.d
#+END_SRC
